VX860

VX680 is a small molecule anti-cancer therapeutic developed by Vertex Pharma. This network shows first and second degree interactions of the molecule which can be searched for understanding its potential downstream effects.

This visualization uses vis.js, and is based on vis.js example 29. Clicking a node grays out all nodes besides the first degree connections to it, and displays the protein names of only the first and second degree nodes.

I've taken node and edge information from string-db for VX680's possible 1st degree connections of ABL1, BCR, Aurora Kinases A-C, and FLT3. This takes all connections above what string-db lists as 0.700 confidence. Thanks to Phil Snyder (Vertex) and Venky Soundararajan (nference) for insightful discussions.
